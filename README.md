# Arsenal Fan Website

A website dedicated to Arsenal Football Club featuring team information, fixtures, player profiles, and club history.



## Technologies Used

- HTML
- CSS


## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/arsenal-website.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## Project Structure

```
arsenal-website/
├── homepage.html
├── fans.html
```
